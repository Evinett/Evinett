# Evinett

**Accelerating OMOP CDM adoption through shared code**

Evinett is a platform for creating and sharing code that supports the use and adoption of the [OMOP Common Data Model (CDM)](https://www.ohdsi.org/data-standardization/the-common-data-model/). It is designed for health data professionals, researchers, and developers to collaborate, share best practices, and accelerate the implementation of standardized health data workflows.

---

## Features

- Share and discover code snippets, ETL scripts, and utilities for OMOP CDM.
- Support collaboration across teams and institutions.
- Improve reproducibility and consistency in health data projects.
- Foster adoption of the OMOP Common Data Model in research and analytics.
